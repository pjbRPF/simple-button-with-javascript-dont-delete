Quantum 01

Qubits and your first quantum gates.

From bits to qubits

A quantum computer uses qubits instead of bits. Like a bit, a qubit has two measurable states, 0 and 1, but as you'll see below, it can do something a bit can't.

The X gate

The X gate is the quantum version of NOT. Set the starting value, choose whether the X gate is present, then measure. Try all four combinations.

The X gate always flips the qubit: 0 → 1 and 1 → 0, every single time. No surprises yet.

The H gate

Now for something a bit can't do. Apply the H (Hadamard) gate and measure the same starting value at least 10 times. Watch what happens.

Take at least 10 measurements to unlock the explanation.
Whichever value you started from, the H gate puts the qubit into an equal superposition of 0 and 1. Every measurement is genuinely random: roughly 50% land on 0 and 50% on 1, but each individual result is still a definite 0 or 1. You never see "half a 0."
